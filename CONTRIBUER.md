# 🤝 CONTRIBUER AU PROJET

Merci de l'intérêt que vous portez à ce panneau de contrôle !

## 📜 Règles d'Or
1.  **Langue** : Toutes les contributions (code, commentaires, docs) doivent être en français.
2.  **Snapshot** : Ne modifiez jamais une version existante. Créez un nouveau dossier de version (ex: `1.0.1`) pour vos changements.
3.  **Souveraineté** : Utilisez des IDs d'entités explicites et francisés.
4.  **Secrets** : N'incluez JAMAIS d'identifiants réels ou secrets. Utilisez des placeholders `A_CHANGER_*`.

## 🛠️ Processus de Développement
- Les fichiers génériques vont dans les dossiers `[Nature]/`.
- Vos fichiers personnels adaptés à votre configuration vont dans `lab/` (et ne seront pas partagés).
